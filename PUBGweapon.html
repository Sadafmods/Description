PUBGMinecraft Addon

This addon is based on a mobile battle royale game called PlayersUnkown Battleground Mobile (PUBG Mobile). This addon will be great for PVP/PVE PUBG Mobile map/project. This addon supported on new version of Minecraft Bedrock Edition

Installation

HOW TO DOWNLOAD MY ADDON? 

  1. Click the link
  2. Click the blue Download button

Done 

HOW TO INSTALL MY ADDON?

  1. Unzip the files
  2. Cut the resource pack file of addon and paste it in Android > Data > com.mojang > resource_pack folder
  3. Do the same with behavior pack but paste it in behavior_pack folder

Done
