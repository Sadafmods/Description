F.R.I.E.N.D. : A Horror Adventure

 

I have been very busy developing the "deluxe update" of friend. As promised, I would upload it on other platforms as well as soon as I had finished the update. This is my first minecraft map, so please do not expect too much from it.

Please know that this is a bedrock edition map!

Plot: You live alone in the deep woods. You wake up on your bed, just like a regular day. But you quickly realize that it's not just a regular day today.

Installation

This map is NOT a virus! For some reason my antivirus marks this map as suspicious while downloading. I don't know if I'm the only one, but I'm writing it here just in case.

Know that this is a bedrock edition map!

Once you downloaded this map, open it by simply clicking on the F.R.I.E.N.D.mcworld file, and minecraft should do the rest. If this is not working, place it in your minecraftWorlds folder, then open it. To find this folder in minecraft for windows, hit the windows key + r, then type %localappdata% and hit enter. Look for a folder named Packages -> Microsoft.MinecraftUWP_8wekyb3d8bbwe -> LocalState -> games -> com.mojang -> minecraftWorlds
